# Torrent Duplication Checker

Інші мови: [English](README.md) · [Русский](README.ru.md)

Настільний застосунок для **Windows** (.NET 10, WPF), який підключається до торрент-клієнта через **Web UI API**, завантажує список торрентів і показує **групи-дублікати**: записи, які ймовірно стосуються одного й того самого релізу за **збігом коментаря** або за **збігом назви й розміру**.

Допомагає побачити зайві торренти та ризикові перетини, коли **одна логічна група дублікатів** має **різні шляхи збереження**.

## Можливості

- **Аналіз** завантажує торренти з клієнта й показує рядки, що належать групі-дублікату (є скасування).
- **Дата для порівняння**: за можливості час створення з метаданих `.torrent` (`creation_date` через API); інакше **added_on** (час додавання в клієнт). Використовується для порядку рядків у групі за однакових шляхів збереження.
- **Кольори рядків**: зелений — найновіший у групі; помаранчевий — старіший (ті самі шляхи збереження); червона підсвітка — у групі **різні шляхи збереження** (підсвічується вся група).
- **Групування** в таблиці: кожна група дублікатів у окремому блоці; заголовок групи — назва торрента.
- **Налаштування**: адреса Web UI, опційно логін/пароль, **мова інтерфейсу** (English / Українська / Русский).
- **Довідка** («?»): коротка легенда кольорів.
- **Відкрити теку збереження**: Провідник відкривається з **виділенням** цієї теки (`explorer /select`).
- Налаштування зберігаються в `%AppData%\TorrentDuplicationChecker\settings.json`.

## Підтримувані торрент-клієнти

| Клієнт | Статус |
|--------|--------|
| [**qBittorrent**](https://www.qbittorrent.org/) (Web UI / API v2) | Підтримується |

**Наразі підтримується лише qBittorrent.** Архітектура дозволяє додати інші клієнти пізніше — наразі вони не реалізовані.

## Вимоги

- Windows  
- [.NET 10](https://dotnet.microsoft.com/download) SDK (для збирання) або runtime (для опублікованих збірок)  
- qBittorrent з увімкненим **Web UI** (і облікові дані, якщо вони використовуються)

## Збірка

```bash
dotnet build -c Release
```

Запускайте `TorrentDuplicationChecker.exe` з каталогу `bin\Release\net10.0-windows\`.
