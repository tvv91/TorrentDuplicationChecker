# Torrent Duplication Checker

Другие языки: [English](README.md) · [Українська](README.uk.md)

Настольное приложение для **Windows** (.NET 10, WPF), которое подключается к торрент-клиенту через **Web UI API**, загружает список торрентов и показывает **группы-дубликаты**: записи, которые с высокой вероятностью относятся к одному и тому же релизу по **совпадению комментария** или по **совпадению имени и размера**.

Помогает заметить лишние торренты и рискованные пересечения, когда **одна логическая группа дубликатов** использует **разные пути сохранения**.

## Возможности

- **Анализ** загружает торренты из клиента и выводит строки, входящие в группу-дубликат (есть отмена).
- **Дата для сравнения**: по возможности время создания из метаданных `.torrent` (`creation_date` через API); иначе **added_on** (время добавления в клиент). Используется для ранжирования строк внутри группы при одинаковых путях сохранения.
- **Цвета строк**: зелёный — самый новый в группе; оранжевый — более старый (одни и те же пути сохранения); красная подсветка — в группе **разные пути сохранения** (подсвечивается вся группа).
- **Группировка** в таблице: каждая группа дубликатов свёрнута в блок; заголовок группы — имя торрента.
- **Настройки**: адрес Web UI, по желанию логин/пароль, **язык интерфейса** (English / Українська / Русский).
- **Справка** («?»): краткая легенда цветов.
- **Открыть папку сохранения**: Проводник открывается с **выделением** этой папки (`explorer /select`).
- Настройки хранятся в `%AppData%\TorrentDuplicationChecker\settings.json`.

## Поддерживаемые торрент-клиенты

| Клиент | Статус |
|--------|--------|
| [**qBittorrent**](https://www.qbittorrent.org/) (Web UI / API v2) | Поддерживается |

**На данный момент реализована поддержка только qBittorrent.** Архитектура допускает добавление других клиентов в будущем — они пока не реализованы.

## Требования

- Windows  
- [.NET 10](https://dotnet.microsoft.com/download) SDK (для сборки) или runtime (для опубликованных сборок)  
- qBittorrent с включённым **Web UI** (и учётные данные, если они используются)

## Сборка

```bash
dotnet build -c Release
```

Запускайте `TorrentDuplicationChecker.exe` из каталога `bin\Release\net10.0-windows\`.
